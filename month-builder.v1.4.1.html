What this builder changes

  • Adds a new month schedule JSON and verification JSON for each imported channel.
  • Updates the imported month list so the imported month becomes the current default for that channel.
  • Keeps older imported months in place so you can still open them from the home page.
  • Builds a full site zip without touching your earlier data.
